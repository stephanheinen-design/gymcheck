# GymCheck 🏋️

Een social check-in app voor de sportschool. Check in als je gaat sporten, zie wie er al is, en compete met vrienden.

## Features
- 📍 Check-in met locatie
- 👥 Vrienden toevoegen
- 🏆 Rankings per week/maand/jaar
- 🥇 Medailles voor weekwinnaars
- 👫 Groepen maken

## Tech Stack
- Backend: Node.js + Express + SQLite
- Frontend: React + Vite
- Deploy: Render.com (gratis tier)

## Deployment

Zie [RENDER-GUIDE.md](../RENDER-GUIDE.md) voor stap-voor-stap instructies.
